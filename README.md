# Irish Waste Management Analysis

Exploratory analysis of quarterly waste generation and treatment patterns across 26 Irish counties.

## Overview
This project analyses two datasets covering waste generation (~19,600 records) and waste treatment (~45,800 records) across Irish counties. The analysis covers waste types, treatment methods, and treatment costs by county and quarter.

## Notebook Structure
- Data loading and inspection
- Standardisation of Quarter format, column naming, and data types
- Missing value handling and outlier treatment
- Merging of both datasets
- Exploratory Data Analysis with visualisations by county, waste type, and quarter
- Findings and conclusions

## Data Sources
- quarterly_waste_generation.csv
- quarterly_waste_treatment.csv

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn
